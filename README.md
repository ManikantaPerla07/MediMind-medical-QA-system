# MediMind 🧠💊

A domain-specific medical question answering system using transformers.

## 🚀 Features

* Multi-type question handling (factoid, list, how, yes/no, comparison)
* Confidence-based abstention (safety mechanism)
* Retrieval + generation hybrid system
* Structured outputs with formatting
* Full-stack deployment

## 🛠 Tech Stack

* Backend: FastAPI, BioBERT, Flan-T5
* Frontend: React (Vite)
* Deployment: HuggingFace + Vercel

## 🌐 Live Demo

Frontend: https://medimind-nu.vercel.app
Backend: https://Manikantaperla-medimind-api.hf.space
